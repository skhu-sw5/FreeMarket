# FreeMarket 프론트엔드

대학생을 위한 중고 거래 플랫폼 FreeMarket의 프론트엔드 프로젝트입니다.

## 시작하기

### 필수 조건
- Node.js 14.0 이상
- npm 6.0 이상

### 설치
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run serve

# 빌드
npm run build
```

## 주요 기능

### 구현 완료 기능
- 사용자 인증
  - 로그인/회원가입
  - 소셜 로그인 (Google, Kakao, Naver)
  - 로그인 유지 기능

- 상품
  - 상품 목록 조회 (필터링, 정렬, 페이지네이션)
  - 상품 상세 조회
  - 상품 등록
  - 관심 상품 등록/취소

- 레이아웃
  - 반응형 디자인 (모바일, 태블릿, 데스크톱)
  - 헤더와 푸터
  - 카테고리 네비게이션

### 구현 예정 기능
- 프로필 페이지
  - 사용자 정보 수정
  - 판매/구매 내역 조회

- 상품 관리
  - 판매 중인 상품 관리
  - 상품 수정/삭제
  - 판매 완료 처리

- 리뷰
  - 거래 후 리뷰 작성
  - 리뷰 조회

- 채팅
  - 판매자와 구매자 간 1:1 채팅

## 프로젝트 구조

```
src/
├── assets/         # 이미지, 스타일시트 등 정적 파일
├── components/     # 재사용 가능한 컴포넌트
│   ├── common/     # 공통 컴포넌트 (헤더, 푸터 등)
│   ├── home/       # 홈 화면 관련 컴포넌트
│   └── products/   # 상품 관련 컴포넌트
├── router/         # Vue Router 설정
├── store/          # Vuex 상태 관리
│   └── modules/    # Vuex 모듈 (인증, 상품 등)
├── views/          # 라우트에 매핑되는 페이지 컴포넌트
│   ├── auth/       # 인증 관련 페이지
│   └── user/       # 사용자 관련 페이지
├── App.vue         # 루트 컴포넌트
└── main.js         # 애플리케이션 진입점
```

## 기술 스택

- **Vue.js 3**: 프론트엔드 프레임워크
- **Vue Router**: 라우팅
- **Vuex**: 상태 관리
- **Tailwind CSS**: 스타일링
- **FontAwesome**: 아이콘
- **Axios** (예정): API 통신

## API 연동

백엔드 API 서버는 `https://freemarket.duckdns.org`에서 호스팅됩니다.

### 알려진 이슈

- **API 연결 문제**: 일부 환경에서 API 연결 오류(ERR_FAILED)가 발생할 수 있습니다. 이는 다음과 같은 방법으로 해결할 수 있습니다:
  1. 브라우저 캐시 및 쿠키 초기화
  2. 브라우저 개발자 도구를 열고 네트워크 탭에서 요청 확인
  3. 필요한 경우 `vue.config.js`의 프록시 설정 확인

- **백엔드 서버 요구사항**: 백엔드 서버가 실행 중이어야 로그인 및 API 기능이 정상 작동합니다.

## 개발 환경 설정 주의사항

1. **프록시 설정**: `vue.config.js` 파일의 프록시 설정을 통해 API 요청이 백엔드 서버로 전달됩니다. API 서버 주소나 포트가 변경된 경우 이 파일을 수정해야 합니다.

2. **로컬 개발 환경**: 기본적으로 Vue 개발 서버는 8081 포트에서 실행됩니다. 백엔드 서버는 기본적으로 8080 포트에서 실행됩니다.

3. **HTTPS 인증서**: 프로덕션 환경에서는 HTTPS를 사용합니다. 개발 환경에서 HTTPS 관련 오류가 발생할 경우 `vue.config.js`의 `secure` 옵션을 `false`로 설정하세요.
